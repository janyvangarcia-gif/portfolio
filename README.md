# Portfolio Website

This is a simple personal portfolio website you can publish on GitHub Pages.

## Customize it
- Replace "Your Name" and the placeholder text in `index.html`
- Update the project cards with your real work
- Change the email and GitHub link

## Publish on GitHub Pages

### Option 1: Project site
1. Create a new GitHub repository.
2. Upload these files to the repository.
3. Open the repository on GitHub.
4. Go to Settings > Pages.
5. Choose the main branch and root folder, then save.

### Option 2: Personal site
If you want your site at `yourusername.github.io`, create a repository named exactly:

`yourusername.github.io`

Then upload these files to the repository root.

## Local preview
Open `index.html` in your browser, or use a simple server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
